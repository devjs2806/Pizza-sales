
SQL is a powerful tool for data analysis,
allowing you to query and manipulate data stored in relational databases. 
In practical scenarios, SQL is used for tasks such as filtering data, aggregating results, joining multiple tables, 
and performing complex calculations.

This is a very basic project showcasing how a database can be made.
The csv files hold various information about different kinds of information about it.
CSV (Comma-Separated Values) file is a plain text format used to store tabular data, where each line represents a data record, and each record's fields are separated by commas. 
CSV files are widely used for data exchange between applications due to their simplicity and compatibility with various software.
After creating the database, information can be extracted out of it.
